# A1-Projeto-geral
Avaliação A1
Este projeto apresenta um sistema simples de cadastro de produtos, permitindo registrar nome, preço e quantidade de cada item. Todas as informações são armazenadas em dicionários, organizados dentro de uma lista principal, o que possibilita cadastrar diversos produtos de forma dinâmica. O programa também exibe um relatório final listando todos os itens registrados e realiza uma análise automática do estoque, identificando quando um produto está sem unidades, com nível baixo ou em quantidade suficiente.
Além das funcionalidades técnicas, o projeto também pode gerar impacto social. Ele pode auxiliar pequenos comerciantes, microempreendedores e projetos comunitários a organizarem melhor seus estoques, evitando desperdícios e facilitando o controle de mercadorias. Isso contribui para o uso consciente de recursos e para a melhoria da gestão em iniciativas sociais, como doações, bazares e distribuição de alimentos. Outro aspecto importante é o incentivo à inclusão digital: por ser um projeto simples, ele permite que iniciantes aprendam lógica de programação e desenvolvam autonomia tecnológica, fortalecendo oportunidades educacionais e profissionais.
